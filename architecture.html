Известные проблемы
🔴
403 на /clients/ endpoint Admin user token не имеет прав на чтение/создание клиентов. Нужно проверить права сотрудника в YCLIENTS или получить токен с полными правами.
🟡
OTP верификация на клиенте Код хранится в localStorage — теоретически можно прочитать с устройства. Для продакшн нужен серверный endpoint проверки.
🟡
Клиент создаётся только при первой записи Из-за 403 клиент не появляется в YCLIENTS при регистрации. Появится автоматически при первом POST /book_record.
🟡
История записей не синхронизируется GET /records/ тоже возвращает 403. История хранится только локально (localStorage).
🟢
Бронирование работает POST /book_record создаёт запись в YCLIENTS и автоматически создаёт клиента если его нет.
