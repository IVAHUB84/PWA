# Release Notes v1.19.3

- **Дата:** 2026-05-30
- **Тип релиза:** patch

## Кратко

Правка тактильного отклика из v1.19.2: убрано масштабирование плашек-контейнеров. `:active` всплывает с вложенной кнопки на контейнер, из-за чего при нажатии кнопки внутри карточки (напр. «Перенести»/«Отменить» в плашке записи) ужималась вся плашка. Теперь отклик только на самих контролах.

## Что нового

### Исправления
- **Убран press-scale + тинт с плашек-контейнеров:** `.svc-catalog-card`, `.master-card`, `.hero-card`, `.feed-card`, `.audience-row`, `.push-template`, `.consent-item`. Раньше нажатие кнопки внутри плашки ужимало всю плашку (всплытие `:active`).
- Отклик нажатия теперь только на: кнопках (`scale` + `brightness`) и мелких контролах — чипы, сегменты, чекбоксы согласия, слоты, табы (`scale`).

### Улучшения
- Убраны ставшие неиспользуемыми токены `--press-bg`, `--press-scale-card`.

## Изменения публичного контракта
- Удалены CSS-токены `--press-bg`, `--press-scale-card` (были добавлены в v1.19.2, в публичном API не использовались).
- Миграции данных не требуется.
