# Release Notes v1.17.0

- **Дата:** 2026-05-29
- **Тип релиза:** minor

## Кратко

Лента студии переведена в разрез реальных категорий и услуг YCLIENTS. Администратор теперь может привязать пост к конкретной услуге или категории каталога, и клиент по кнопке «Записаться» попадает напрямую на карточку услуги или в отфильтрованный каталог — без тупиков «Ничего не найдено».

## Что нового

### Возможности
- Три типа поста: **Свободный** (без привязки), **По категории**, **По услуге**. Выбор в админ-форме.
- Для типа «По услуге»: поиск по реальному каталогу YCLIENTS, выбор фиксирует `serviceId`/`serviceName`/`cat`.
- Для типа «По категории»: селект реальных категорий из уже загруженных данных (`SERVICES_DATA`).
- CTA «Записаться →» на посте по услуге открывает карточку именно этой услуги (`s-service`).
- CTA на посте по категории открывает каталог, отфильтрованный по этой категории.
- Фильтр ленты по типу поста: **Все** / **Акции и услуги** / **Новости**.

### Улучшения
- Graceful fallback: если привязанная услуга/категория удалена из YCLIENTS — CTA молча открывает общий каталог без ошибок в консоли.
- Старые посты (только поле `cat`, без `type`) трактуются как «Свободные» без миграции данных.
- Подпись в списке постов админки отображает название услуги, категорию или тип.

### Исправления
- CTA «Записаться» у свободного поста скрыт (ранее всегда вёл в каталог, мог давать «Ничего не найдено»).
- Теги карточки поста с пустым `cat` (свободный пост) больше не рендерятся как пустой элемент.

## Изменения публичного контракта

**`posts.json` (GitHub-синхронизация):** добавлены необязательные поля `type` (`'free'|'category'|'service'`), `serviceId`, `serviceName`. Поле `cat` сохранено обязательным (для `free` = `''`). Изменение обратносовместимо: посты без `type` читаются как `free`; валидатор `_VALID_POST` не изменён.

Миграция существующих постов не требуется.

## Известные ограничения
- Словари градиентов/иконок в `feed.js` и `profile.js` не покрывают все реальные категории YCLIENTS (например `Губы`, `Глаза`, `Эпиляция`). Для незнакомых категорий отображается generic-плейсхолдер (`📝` / серый градиент) — это не регрессия, расширение словарей отложено как техдолг.
- Пост по мастеру, несколько услуг в одном посте, промо-бейдж со скидкой — вне scope v1.17.0 (отдельные требования).
