# Release Notes v1.19.2

- **Дата:** 2026-05-30
- **Тип релиза:** patch

## Кратко

Усилен тактильный отклик нажатия из v1.18.0 — эффект сделан заметнее. Карточки, слоты и чипы при нажатии не только уменьшаются сильнее, но и подсвечиваются тёплым брендовым тинтом. Все кнопки (включая инлайн-стилевые: «Записать себя/другого», «Перенести», «Отменить») получили единый отклик через `scale` + затемнение `brightness`.

## Что нового

### Улучшения
- **Усилен scale press feedback:** карточки `0.97 → 0.94`, чипы `0.96 → 0.93`, карточка мастера `0.99 → 0.94` (через токен).
- **Смена фона на `:active`** для карточек услуг/мастеров, `.hero-card / .feed-card / .audience-row / .push-template / .consent-item / .slot`, чипов — новый токен `--press-bg: var(--accent-light)` (тёплый тинт, автоматически адаптируется к тёмной теме).
- **Универсальный отклик для всех `<button>`** — `transform: scale(0.93)` + `filter: brightness(0.88)`. `brightness` работает поверх любого фона, в т.ч. заданного инлайн-стилем, поэтому реагируют и кнопки без CSS-класса. Исключения: FAB и табы (свой отклик).
- Новые токены: `--press-bg`, `--press-brightness`.

### Исправления
- Кнопки с инлайн-фоном (`Записать себя/другого`, `Перенести`, `Отменить` в истории) ранее не давали видимого отклика на нажатие — теперь дают.

## Изменения публичного контракта
- CSS-токены `--press-bg`, `--press-brightness` — добавлены в `:root`. Обратная совместимость сохранена.
- Миграции данных не требуется.

## Известные ограничения
- `prefers-reduced-motion` отключает `scale`, `filter` и pulse — отклик у таких пользователей минимален (по дизайну).
- Финальное визуальное подтверждение — в браузере.
