# Release Notes v1.16.0

- **Дата:** 2026-05-29
- **Тип релиза:** minor

## Кратко

На экране выбора мастера убран наложенный значок-сердечко (❤️-кружок) с аватарки избранного мастера. Аватар избранного теперь выглядит так же, как у обычного мастера. Все остальные признаки избранности (текстовая метка «Избранная», кнопка-переключатель ❤️/🤍, рамка-акцент карточки, сортировка вверх, хранение в `localStorage`) сохраняются без изменений.

## Что изменено

### Улучшения

- **Убран значок-сердечко с аватарки избранного мастера в списке (BA-021, ADR-018).**
  - Из `docs/modules/masters.js` (`_masterCardHtml`) удалены формирование переменной `favBadge` и её интерполяция `${favBadge}` в блоке аватара (`masters.js:12,24`). Элемент `<div class="fav-badge">❤️</div>` в DOM карточки мастера больше не рендерится.
  - Из `docs/style.css` удалено неиспользуемое после правки CSS-правило `.fav-badge { ... }` (`style.css:320`). Мёртвого кода не осталось.
  - Сохранены все прочие признаки избранности: метка «Избранная» (`fav-chip`), кнопка-переключатель ❤️/🤍 (`toggleFav` в карточке списка и в детальной карточке), рамка-акцент `master-card.fav`, сортировка избранных вверх, хранение `yc_favs` в `localStorage`.

### Служебное

- `CACHE_VERSION` поднята `v64 → v65` (изменены файлы `APP_SHELL`: `masters.js`, `style.css`).
- `docs/version.json`: build 31 → 32, version v1.16.0.

## Изменения публичного контракта

Внешний API / YCLIENTS / схемы данных не меняются. Правка чисто фронтовая, состояние избранного остаётся локальным.

Внутренний клиентский контракт (`localStorage` `yc_favs`, поведение `toggleFav`, формат записи `yc_reviewed_ids`) не меняется. Новых ключей хранения нет — миграция не требуется.

Наблюдаемое изменение UI: в списке выбора мастера у избранных мастеров исчезает значок ❤️ в правом нижнем углу аватара. Все остальные элементы экрана мастеров, а также прочие экраны (главная, услуги, слоты, история, профиль) — не затронуты.

## Известные ограничения

- Проверка в реальном браузере (сценарии С1–С6: рендер аватаров с фото и инициалами, переключение избранного, DevTools → SW-кэши `studio-static-v65` / `studio-runtime-v65`, Lighthouse installable) требует ручного прогона после деплоя.
