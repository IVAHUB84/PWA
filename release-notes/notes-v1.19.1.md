# Release Notes v1.19.1

- **Дата:** 2026-05-30
- **Тип релиза:** patch (фикс карусели фото услуг)

## Кратко

Хотфикс: горизонтальная карусель фото на детальной карточке услуги (`s-service`) не реагировала на мышь и не имела активных точек-индикаторов. Перевели управление на Pointer Events — теперь работают и тач-свайп, и drag мышью; точки-индикаторы стали кликабельными.

## Что изменено

### Карусель (`docs/modules/services.js`)

- Заменены `touchstart/touchmove/touchend` на `pointerdown/pointermove/pointerup/pointercancel` — единый код для тача, мыши и пера.
- Добавлен live drag-follow: трек следует за пальцем/курсором во время свайпа, на отпускании — snap к ближайшему слайду либо переход по направлению.
- Ось определяется первыми 8 px движения; вертикальный жест сразу отпускается родительскому скроллу.
- Порог срабатывания свайпа: 40 px по горизонтали.
- `setPointerCapture` гарантирует получение `pointerup`, даже если курсор уходит за пределы трека.
- Точки `.svc-carousel-dot` стали кликабельными — клик переводит на выбранный слайд (через `goTo`, без сброса бесконечной петли).
- Подавление click-through после реального drag, чтобы не открывать карточку случайно.

### CSS (`docs/style.css`)

- `.svc-carousel-track`: `touch-action: pan-y` (горизонтальный жест — наш; вертикальный отдан странице), `cursor: grab` / `:active` `grabbing`, `user-select: none`.
- `.svc-carousel-slide img`: `pointer-events: none` + `user-drag: none` — изображение не перехватывает pointerdown и не запускает native drag-and-drop в десктопных браузерах.
- `.svc-carousel-dot`: `cursor: pointer`.

### Служебное

- `CACHE_VERSION` поднята `v68 → v69` (изменены `services.js` и `style.css` из `APP_SHELL`).
- `docs/version.json`: `build` 35 → 36, `version` → `v1.19.1`.

## Изменения публичного контракта

Нет. Внешний вид галереи (высота 280 px, peek, точки, скругления, тень), формат данных `s.photos`, навигация, бесконечная петля с клонами — без изменений.

## Проверка

- **Lint:** `npm run lint` — чисто.
- **Tests:** `npm test` — 42 прошло.
- **Ручная проверка:** на услуге с ≥2 фото — свайп пальцем (мобайл), drag мышью (десктоп), клик по точкам; бесшовный переход «последний ↔ первый»; одно фото / нет фото — без регрессий.
