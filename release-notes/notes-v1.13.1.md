# Release Notes v1.13.1

- **Дата:** 2026-05-29
- **Тип релиза:** patch (UI-доработки инбокса уведомлений)

## Кратко

Хотфикс по замечаниям к v1.13.0: бейдж непрочитанных перенесён в правый нижний угол колокольчика (как в исходном требовании), очистка истории вынесена в иконку-веник в шапке инбокса, уведомления отображаются отдельными карточками-плашками.

## Что изменено

### UI

- **Бейдж непрочитанных — справа снизу** колокольчика (`style.css` `.inbox-badge`: `top` → `bottom`). Соответствует исходному требованию req-016.
- **Иконка-веник для очистки** — в правом верхнем углу шапки экрана `s-inbox` (`prototype.html`, кнопка `.inbox-clear-btn` с `aria-label`, вызывает `clearInboxHistory()`). Прежняя текстовая кнопка «Очистить историю» внизу списка удалена.
- **Уведомления — карточки-плашки** (`style.css` `.inbox-item`): фон `--surface`, скругление, отступы и лёгкая тень вместо плоской строки с разделителем.

### Служебное

- `CACHE_VERSION` поднята `v60 → v61` (изменены `prototype.html` и `style.css` из `APP_SHELL`).
- `docs/version.json`: build 27 → 28, version v1.13.1.

## По диагностике «мерцания» колокольчика

«То появлялся, то нет» — это разовая авто-перезагрузка страницы при смене Service Worker (`app.js` `controllerchange` → `location.reload()`), сработавшая при переходе v59 → v60 сразу после деплоя v1.13.0. Это штатное поведение обновления PWA; на стабильной версии не повторяется. Логика обновления SW не менялась. После установки v61 произойдёт ещё одна разовая перезагрузка — это ожидаемо.

## Изменения публичного контракта

Нет. Доставка push, CF Worker, схема подписок (ADR-006), навигация и формат хранилища IndexedDB `studio-inbox` не менялись.

## Проверка

- **Lint:** `npm run lint` — чисто.
- **Изменения только в разметке/стилях** (`prototype.html`, `style.css`) + `CACHE_VERSION`/`version.json`; JS-логика инбокса не затронута.
- **Требует визуальной проверки в браузере:** позиция бейджа (низ-право), иконка-веник в шапке инбокса и очистка по ней, отображение уведомлений плашками. Проверить после обновления Service Worker (hard-reload).
