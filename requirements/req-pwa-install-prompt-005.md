# Требование 005: Предложение установить приложение на экран «Домой»

## Описание
При открытии приложения на мобильном устройстве (если оно ещё не установлено как PWA) — автоматически показывать предложение добавить ярлык на экран «Домой».

## Ожидаемое поведение
- Пользователь открывает сайт в браузере на смартфоне
- Появляется баннер/подсказка с предложением установить приложение
- Пользователь может принять предложение или отклонить
- Если пользователь уже установил приложение (standalone mode) — ничего не показывать
- Если пользователь отклонил — не показывать повторно (или показывать раз в N дней)

## Платформы
- Android / Chrome — программный prompt через `beforeinstallprompt`
- iOS / Safari — нет программного API, нужна ручная инструкция ("нажмите «Поделиться» → «На экран «Домой»»")
- Другие браузеры — по возможности

## Дополнительно
- Показывать сразу при открытии (не ждать взаимодействия)
- Дизайн — в стиле приложения (не нативный браузерный баннер)
